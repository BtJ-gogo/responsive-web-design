HTML要素の基礎

  • html要素
  • head要素
  • title要素
  • link要素
  • meta要素

html要素

ルート要素を表す。

主な属性

  • lang

使用例

<html lang="en"></html>

head要素

そのページのメタデータを表す。

使用例

<head>   <title>HTML文章のタイトル。</title> </head>

title要素

HTML文章のタイトルを表す。

使用例

<title>HTML文章のタイトル。</title>

link要素

文章を他の外部リソースと関連付ける。

主な属性

  • href
  • rel

使用例

<link rel="stylesheet" href="styles.css" />

meta要素

メタデータの具体的な項目を表す。

主な属性

  • charset
  • name
  • content

使用例

<meta charset="utf-8" /> <meta name="viewport" content="width=device-width, initial-scale=1.0" />

※必ずしも正しいとは限りません。重要な情報は確認するようにしてください。
